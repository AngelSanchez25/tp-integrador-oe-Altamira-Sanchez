## Descripción del Proyecto

Este proyecto consiste en la automatización del proceso de solicitud de vacaciones de empleados mediante un chatbot desarrollado en Python.

El sistema permite que un empleado ingrese su DNI, consulte la información almacenada en una base de datos simulada (Excel) y obtenga una respuesta automática sobre la aprobación o rechazo de su solicitud.

## Objetivo

Aplicar los conceptos vistos en la materia Organización Empresarial mediante el análisis de un proceso organizacional y su posterior automatización.

## Tecnologías Utilizadas

* Excel

## Estructura del Proyecto

* chatbot.py → Código fuente del chatbot.
* empleados.xlsx → Base de datos simulada de empleados.
* bpmn_as_is.png → Proceso actual (AS-IS).
* bpmn_to_be.png → Proceso mejorado (TO-BE).
* informe_final.pdf → Documentación completa del proyecto.

## Funcionamiento

1. El usuario ingresa su DNI.
2. El sistema consulta la base de datos.
3. Se verifica la cantidad de días disponibles.
4. El chatbot aprueba o rechaza la solicitud.
5. Se informa el resultado al usuario.

## Casos Contemplados

* Solicitud aprobada.
* Solicitud rechazada.
* DNI inexistente.
* Error de ingreso de datos.
